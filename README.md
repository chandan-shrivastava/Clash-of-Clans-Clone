# Clash of Clans Clone

## How to run:

```bash
python3 main.py
```

## How to play:

```
w = move up
a = move left
s = move down
d = move right
h = heal spell
r = rage spell
i,o,p = spawn barbarians
q = quit
```

## Replay
### Plays the replay of last game. Here number is the number of game (0 indexed).

```bash
python3 replay.py <number>
```
